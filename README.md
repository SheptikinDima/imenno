# ИМЕННО

Адаптивный лендинг бренда персонализированных подарков на Vite, React и
TypeScript. Внутри: обновленный визуал по PDF-макету, Framer Motion-анимации,
форма заявки и serverless API.

## Запуск

```bash
npm install
npm run dev
```

Для локальной проверки serverless API используйте `vercel dev`. Обычный
`vite` запускает только клиентскую часть.

## Форма

- поля: имя, телефон, e-mail, согласие;
- клиентская и серверная валидация;
- honeypot-поле против ботов;
- rate limit: 5 запросов с одного IP за 10 минут;
- Upstash Redis для распределенного rate limit в production;
- доставка заявок через webhook и/или Resend.

Настройки перечислены в `.env.example`.

## Шрифт

В стилях указан локальный `Angst`. Если файл шрифта установлен в системе, сайт
использует его автоматически. Если нужно подключить webfont-файл, добавьте его в
проект и обновите `@font-face` в `src/styles.css`.
